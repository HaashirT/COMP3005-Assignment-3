Setup Instructions:
1) Create the new database in pgAdmin4
2) Use the provided setup.sql file inside the database folder to create the database's schema and insert the initial values
3) Update DB_CONFIG in application.py inside the application folder to match your database's connection details
4) Install psycopg2 by typing "pip install psycopg2" in the terminal
4) Run the application
